Building of the pan language compiler and the associated documentation relies
on the following software packages.  Use of these packages is gratefully
acknowledged. 

To build the software from this source distribution the external packages
will have to be downloaded and installed in the external directory.  
The subdirectory name, version, and URL are listed below.

apache-ant (1.7.0): http://ant.apache.org/

docbook (5.0): http://www.docbook.org/schemas/5x.html

docbook-xsl-ns (1.74.0): http://sourceforge.net/projects/docbook/

findbugs 1.3.4: http://findbugs.sourceforge.net/

fop (fop-0.95beta): http://xmlgraphics.apache.org/fop/

javacc (4.0): https://javacc.dev.java.net/

junit (4.5): http://www.junit.org/

offo-hyphenation-fop (1.0): http://offo.sourceforge.net/

saxon6-5-5 (6.5.5): http://saxon.sourceforge.net/
